# 🚀 v1.0.0 - Professional MAC Address Spoofer

A professional-grade, production-ready CLI tool for network interface MAC address spoofing on Linux systems.

### What's New
- 🛡️ **Stealth Operations**: Clear terminal history and run silently for automation.
- 🏢 **Vendor Spoofing**: Look like a Samsung, Apple, or Google device with one flag.
- 🔄 **Auto-Rotation**: Keep your identity moving with the new `--interval` daemon mode.
- 💾 **MAC Profiles**: Save your favorite configurations and reload them instantly.
- 🖥️ **Interactive TUI**: A beautiful, menu-driven interface for those who prefer menus over flags.
- ⚙️ **Systemd Integration**: Persistent spoofing that survives reboots.

### Installation

**Using pipx (Recommended):**
```bash
pipx install git+https://github.com/joemunene-by/mac-spoofer.git
```

**From Source:**
```bash
git clone https://github.com/joemunene-by/mac-spoofer.git
cd mac-spoofer
sudo make install
```
